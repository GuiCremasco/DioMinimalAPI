# DioMinimalAPI

Este repositório possui um projeto de Minimal API no ASP.NET Core 10, cuja premissa é uma API com operações CRUD com dados de veículos. O projeto é parte do módulo **"Trabalhando com ASP.NET Minimals APIs"** do curso de Backend com .NET ministrado pela DIO.

*This repository contains an ASP.NET Core 10 Minimal API project whose purpose is to provide a CRUD API for vehicle data. The project is part of the **"Working with ASP.NET Minimal APIs"** module of DIO's .NET Backend course.*
